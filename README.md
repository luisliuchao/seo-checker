# SEO Checker

A simple Node.js package to let a user use this package to scan a HTML file and show all of the SEO defects. [What is SEO?](https://moz.com/learn/seo/what-is-seo)

## Requirements

- yarn

## Install

```bash
  yarn add seo-checker
```

## Test

```bash
  yarn test
```

## Build

```bash
  #  for development
  yarn dev

  #  for production
  yarn prod
```
